## Development
- Ruby
    - Use [`RVM: Ruby Version Manager`](https://rvm.io/)
    - `rvm install 2.1`
    - `rvm use 2.1 --default`
    - `ruby --version` > 2.1.0
- Ruby dependencies
    - `gem install bundler`
    - `bundle install`
        - Installs dependencies listed in the `Gemfile`
- Local site preview
    - `bundle exec jekyll serve`
    - [`http://localhost:4000`](http://localhost:4000)
- Keep dependencies up-to-date with `bundle update`
